# 🏆 Gestor de Torneos (Single-File App)

Una aplicación web ultra-ligera y completamente funcional contenida en un único archivo `.html`. Diseñada para organizar torneos rápidos (Smash Bros, FIFA, juegos de mesa, etc.) con amigos, sin dolores de cabeza.

✨ **Características principales:**
* **Plug & Play:** Sin instalaciones, sin Node.js, sin dependencias. Haz doble clic en el archivo y listo.
* **Brackets Dinámicos:** Genera cuadros de torneos perfectos matemáticamente (con pases directos automáticos si sois impares).
* **Guardo Local Segura:** Guarda el progreso de tus torneos múltiples directamente en el almacenamiento de tu navegador. Si cierras la ventana, no pierdes los datos.
* **Doble Eliminación:** Soporte opcional para cuadro de repesca (Losers Bracket) y gran final con "Muerte Súbita".
* **Completamente Offline:** Una vez descargado el archivo, no necesitas conexión a internet para usarlo.

🚀 **¿Cómo usarlo?**
Simplemente descarga el archivo `Torneo.html` y ábrelo en cualquier navegador web (Chrome, Firefox, Safari...). ¡A jugar!
